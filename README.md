# Paginas Web Showcase

Coleccion publica de portafolio para mostrar distintos sitios y micrositios desarrollados, sin publicar el codigo completo de produccion.

## Casos incluidos

### OPTIMANIA

Sitio web comercial para optica con catalogo, producto, carrito, pago, promociones, blog, usuarios y panel administrativo.

Tecnologias principales:
- PHP
- HTML
- CSS
- JavaScript

Modulos visibles por estructura del proyecto:
- catalogo y detalle de productos
- carrito y flujo de pago
- promociones y contenido informativo
- registro, login y usuarios
- panel administrativo y gestion de pedidos

### MYCABO

Sitio de cotizacion y reserva orientado a traslados y hospedaje, con secciones de booking, hotel, unidades, FAQ, buscador y contenido SEO.

Tecnologias principales:
- PHP
- HTML
- CSS
- JavaScript

Capacidades destacadas:
- cotizacion y reserva
- paginas de hotel y unidades
- buscador y contenido FAQ
- rutas SEO y sitemap
- panel administrativo

### OPTIMANIA maybe

Micro sitio/demo en Flask con APIs simples para productos, testimoniales y sucursales.

Tecnologias principales:
- Python
- Flask
- HTML
- CSS
- JavaScript

Capacidades destacadas:
- landing comercial
- endpoints JSON para frontend
- datos de productos, testimoniales y sucursales

## Enfoque de portafolio

Este repositorio sirve para mostrar variedad de trabajo web: e-commerce, contenido corporativo, cotizacion/reservas y micrositios con backend ligero.

## Alcance publico

Se publica solo la documentacion de showcase. El codigo de produccion, configuraciones privadas, integraciones reales, datos sensibles y despliegues no se exponen aqui.
